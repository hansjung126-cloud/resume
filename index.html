성취를 동력으로 삼는 마케터
질문만 던지면 AI가 인사이트를 도출하고 디자인을 생성하는 시대에, 마케터의 역할은 이러한 기술을 실무에 얼마나 효과적으로 녹여내는지에 달려 있다고 생각합니다. 저는 퍼포먼스 마케팅의 기본기를 충실히 다지는 것은 물론, AI와 자동화 툴을 활용해 업무 효율을 높이는 데 많은 관심을 기울여 왔습니다.

어려운 문제를 마주했을 때 이를 끝내 해결해 내며 얻는 성취감이 제게는 가장 큰 동기부여입니다. 스스로 한계를 두지 않고 몰입하여 얻은 성과가 회사의 성장으로 이어지는 모두닥의 치열한 환경 속에서, 2026년 목표 매출 300억 달성에 보탬이 되는 든든한 일원으로 성장하고 싶습니다.
데이터 이면의 고객 행동을 읽다
커머스 브랜드를 담당할 당시, 채널별 판매 데이터를 분석하다 흥미로운 패턴을 발견했습니다. 진입 장벽이 낮은 스마트스토어는 '샴푸' 단일 구매 비중이 높았지만, 자사몰은 '트리트먼트' 병행 구매 비중이 눈에 띄게 높았습니다.

"네이버에서 샴푸를 써본 고객이 만족하면 혜택이 많은 자사몰로 유입되어 트리트먼트를 추가 구매할 것"이라는 가설을 세웠습니다. 샴푸 구매 유저와 유사 타겟을 설정하고 맞춤 소재를 기획한 결과, ROAS 206%라는 유의미한 성과를 거두었습니다. 고객의 여정에서 힌트를 얻고 타겟팅을 연결해 본 경험으로 모두닥의 실험 사이클에 빠르게 적응하겠습니다.
[필수 문항 1] 현재 가장 집요하게 매달리는 목표

더 나은 사용 환경을 향한
백엔드 서버 구축 도전

현재 저는 기존에 구축해 둔 '데이터 자동화 웹 대시보드'의 초기 로딩 속도를 개선하기 위해, 백엔드 서버를 직접 공부하며 구축하는 프로젝트에 매달리고 있습니다. 과거 엑셀 리포트를 웹으로 전환하여 실시간 데이터 시각화에는 성공했지만, 방대한 구글 시트 데이터를 프론트엔드 단에서 처리하다 보니 데이터가 누적될수록 로딩 지연 현상이 발생했습니다.

'마케터로서 이 정도면 충분하지 않을까' 스스로 타협할 수도 있었습니다. 하지만 데이터 확인의 직관성과 속도는 대시보드를 사용하는 분들에게 가장 중요한 경험이기에, 이를 방치하고 싶지 않았습니다. 비록 직무 경계를 넘어서는 낯선 영역일지라도, 문제의 본질을 파악하고 기술적인 해결책을 찾아 부딪혀보는 저의 끈기는 헬스케어 정보의 비대칭성을 해결해 나가는 모두닥의 비전과 잘 어울린다고 생각합니다.
[필수 문항 2] 회의적 시선을 끝까지 밀고 나간 경험

현실적인 제약 속에서 찾은
새로운 성장의 발판

"현재 부서의 리소스와 시간 부족으로 새로운 대시보드를 당장 도입하는 것은 현실적으로 어렵습니다." — 당시 팀장님 피드백

대행사에서 근무하며, 실무자에게만 익숙하고 광고주 입장에서는 인사이트를 파악하기 힘든 기존 엑셀 리포트 양식을 개선하고자 대시보드 개편안을 기획했습니다. 팀장님의 만류로 주어진 업무에만 집중할 수도 있었지만, 기존 방식을 조금이라도 개선해보고 싶은 마음에 퇴근 후 개인 시간을 할애하여 피벗테이블과 수식을 활용한 데이터 구조화 작업에 몰두했습니다.

결과적으로 실무 도입은 무산되었지만, 이 과정은 제게 결코 실패가 아니었습니다. 오히려 '기존 툴 안에서 시각화만 개선할 것이 아니라, 데이터 수집 자체를 자동화하는 시스템을 공부해보면 어떨까'라는 새로운 목표를 갖게 해 주었고, 이는 이후 웹 대시보드 자동화 프로젝트를 시작하는 든든한 밑거름이 되었습니다.
[필수 문항 3] 가장 도전적인 목표와 몰입의 결과

하루 3시간, 3개월의 꾸준함
10분의 효율화가 가져온 성과

1
낯선 기술 환경으로의 몰입
개발 지식은 부족했지만, '데이터 취합 100% 자동화'를 위해 3개월간 매일 퇴근 후 3시간씩 몰입했습니다. Make.com을 활용해 오전 9시 자동화 트리거를 설정하고, 까다로운 네이버 API 서명(Signature) 인증과 HTTP 모듈 세팅을 거쳐 데이터를 성공적으로 수집 및 연동했습니다.
2
반복 업무 10분 단축, 그리고 가설 검증
광고주 1명당 매일 15분 소요되던 취합 작업이 5분으로 단축되었습니다. 확보한 10분의 여유를 데이터 분석에 온전히 투자하여 기존 세트 구조를 재점검했습니다. 머신러닝 효율을 높이기 위해 핵심 소재를 집중시키고 공격적인 A/B 테스트를 지속 운영했습니다.
✓
ROAS 46% 개선 달성
단순 반복 업무를 줄이고 마케터 본연의 고민에 집중한 결과, 담당 광고주의 ROAS를 46% 개선하는 유의미한 성과를 거두었습니다. 낯선 허들 앞에서도 끈기 있게 몰입하여 실무에 적용해 본 이 경험을 모두닥에서도 증명해 보이겠습니다.
